📚 Về Lộ trình 1000 từ vựng TOEIC

Trang web đã được nhúng sẵn 1000 từ vựng TOEIC thông dụng nhất. Hệ thống tự động xáo trộn ngẫu nhiên và chia thành 34 bài học (30 từ/bài). Cấu trúc trộn này được cố định nên bài học của bạn sẽ không bị nhảy chữ dù bạn có làm mới trang.

💾 Dữ liệu được lưu ở đâu?

Vì đây là trang web chạy trên Github Pages, tiến độ học của bạn được lưu tự động và hoàn toàn trực tuyến trên trình duyệt của máy tính/điện thoại bạn đang dùng. Web không cần đăng nhập, hoàn toàn miễn phí và không ai có thể truy cập dữ liệu của bạn.

🔄 Làm sao để chuyển máy tính/điện thoại vẫn giữ nguyên bài học?

Rất đơn giản, bạn chỉ cần bấm vào nút Cài đặt (Hình bánh răng) -> Chọn "Xuất dữ liệu". Trang web sẽ tải về một file nhỏ (backup.json). Hãy gửi file này sang máy tính mới, mở web lên và bấm "Nhập dữ liệu" để tiếp tục học.

🔊 Về hệ thống phát âm

Hệ thống sử dụng cơ chế kéo file âm thanh MP3 cực chuẩn (Giọng Anh-Mỹ) từ máy chủ Google Translate & Từ điển Youdao. Vì vậy, phát âm sẽ giống hệt nhau trên tất cả các thiết bị (iPhone, Android, Windows, Mac) mà không phụ thuộc vào hệ điều hành.

Cài đặt & Dữ liệu

Trang web lưu tiến độ Offline. Hãy bấm Xuất dữ liệu để tải file backup, gửi qua máy khác rồi chọn Nhập dữ liệu để học tiếp tiến độ hiện tại.

Tính năng này sẽ tạo mới toàn bộ tiến trình: lấy lại 1000 từ vựng gốc, xáo trộn thứ tự mới và chia lại vào các bài học. Tiến độ cũ sẽ bị mất.
